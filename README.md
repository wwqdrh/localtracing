## 背景

单体应用的链路运行情况记录

## 特性

- 🗂 调用链路耗时分析
- 📦 P99分为

## 使用手册

### 使用
<br>

```go
engine := gin.Default()
hand, err := localtracing.NewMonitor(&GinHanlderAdapter{
    engine: engine,
}, "./logs")
engine.Use(hand.HandlerFunc())
```

实时日志查看: http://localhost:8080/view?file=log.txt

在对应的log.txt新建日志记录查看效果
